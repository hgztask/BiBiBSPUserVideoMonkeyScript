# b站屏蔽增强器v2版

<hr>
对B站内容进行屏蔽，如视频、评论、动态、直播间的评论等，详情可看下面支持的屏蔽类型

- 脚本尽量减少对B站布局的修改和样式修改
- 支持快捷屏蔽按钮，鼠标放在视频标题或评论上会显示屏蔽按钮，点击可选择uid精确屏蔽和用户名精确屏蔽，建议优先uid方式
- 所有屏蔽规则都可以在主面板，快捷键~打开或点击页面左上角按钮展开，中的规则管理按需添加规则
- 脚本发布在[脚本猫](https://scriptcat.org/zh-CN/script-show-page/1029/)
  和[greasyfork](https://greasyfork.org/zh-CN/scripts/461382)平台，脚本更新会第一时间同步到两个平台
- 哔哩哔哩增强屏蔽器常见问题汇总(
  持续更新)[https://docs.qq.com/doc/DSlJNR1NVcGR3eEto](https://docs.qq.com/doc/DSlJNR1NVcGR3eEto)
- 具体功能可看下面的相关描述
- 欢迎提出优化建议和需求，用户的赞助支持就是维护更新的最大动力，最下方有赞助码

## 屏蔽的类型

| 功能描述        | 备注         | 功能描述          | 备注         |
|-------------|------------|---------------|------------|
| 用户名模糊匹配     |            | 标题模糊匹配        |            |
| 用户名精确匹配     |            | 标题正则匹配        |            |
| 用户名正则匹配     |            | 评论模糊匹配        |            |
| 用户uid匹配     |            | 评论正则匹配        |            |
| 用户白名单uid匹配  |            | 话题tag精确匹配     |            |
| 话题tag模糊匹配   |            | 话题tag正则匹配     |            |
| 粉丝牌精确匹配     |            | 直播分区黑名单(精确匹配) |            |
| 视频最短时长屏蔽    | 低于设置值的会屏蔽  | 视频最长时长屏蔽      | 高于设置值的会屏蔽  |
| 视频最大弹幕量屏蔽   | 高于设置值的会屏蔽  | 视频最小弹幕量屏蔽     | 低于设置值的会屏蔽  |
| 视频最小播放量屏蔽   | 低于设置值的会屏蔽  | 视频最大播放量屏蔽     | 高于设置值的会屏蔽  |
| 评论区最小用户等级过滤 | 低于设置值的会屏蔽  | 评论区最大用户等级过滤   | 高于设置值的会屏蔽  |
| 视频tag标签模糊匹配 |            | 视频tag标签精确匹配   |            |
| 视频tag标签正则匹配 |            | 专栏屏蔽          | 暂不支持       |
| 热搜关键词模糊匹配   | 页面顶部搜索框的热搜 | 热搜关键词正则匹配     | 页面顶部搜索框的热搜 |

- 需要注意的是模糊匹配和正则匹配的规则不用考虑要匹配的内容大写问题，如标题，会自动转成小写进行匹配
- 视频tag不包括视频的分区，比如鬼畜

| 最大视频时长屏蔽   | 最小视频弹幕量屏蔽   | 最小视频播放量屏蔽   | 最大视频弹幕量屏蔽   |
| 最大视频播放量屏蔽  | 视频tag标签模糊匹配 | 视频tag标签正则匹配 | 视频tag标签精确匹配 |

| 用户名模糊匹配 | 用户名精确匹配     | 用户名正则匹配     |
|---------|-------------|-------------|
| 用户uid匹配 | 用户白名单uid匹配  | 评论模糊匹配      |
| 评论正则匹配  | 评论区最小用户等级过滤 | 评论区最大用户等级过滤 |

### 用户个人动态页中的动态屏蔽类型

|--------------|-----------|--------------|
| 评论正则匹配(动态内容) | 话题tag精确匹配 | 话题tag模糊匹配    |
| 话题tag正则匹配    |

### 直播间内评论的屏蔽类型

|---------|------------|---------|
| 用户uid匹配 | 用户白名单uid匹配 | 评论模糊匹配  |
| 评论正则匹配  | 粉丝牌精确匹配    |         |

### 直播房间列表的屏蔽类型

| 直播分区黑名单(精确匹配) | 用户uid匹配 | 用户白名单uid匹配 |
|---------------|---------|------------|
| 用户名模糊匹配       | 用户名精确匹配 | 用户名正则匹配    |
| 标题模糊匹配        | 标题正则匹配  |            |

## 处理的页面和位置

### 首页(https://www.bilibili.com/)

1. 首页换一换视频列表
2. 首页换一换视频列表下面的视频列表

### 搜索页(https://search.bilibili.com/*)

1. 搜索页中的综合选项卡和视频选项卡中的视频列表

### 视频播放页(https://www.bilibili.com/video/*)

1. 底下的评论区评论
2. 右侧的推荐视频列表

### 合集视频播放页(https://www.bilibili.com/list/ml*)

1. 底下的评论区评论
2. 右侧的推荐视频列表

### 稍后再看视频播放页(https://www.bilibili.com/list/watchlater*)

1. 底下的评论区评论
2. 右侧的推荐视频列表

### 动态页(https://t.bilibili.com/)

1. 动态页评论区评论

### 动态详情页(https://www.bilibili.com/opus/*)

1. 底下的评论区评论
2. 包括旧版的评论区评论页

### 话题详情页(https://www.bilibili.com/v/topic/detail/*)

1. 话题详情页评论区评论
2. 话题详情页中视频项
3. 话题详情页中动态项

### 番剧或影视播放页(https://www.bilibili.com/bangumi/play/*)

1. 底下评论区的评论

### 热门页(https://www.bilibili.com/v/popular/*)

1. 综合热门中的视频列表(该页面不支持使用uid、时长屏蔽，其他的不受影响)
2. 每周必看中的视频列表(该页面不支持使用uid、时长屏蔽，其他的不受影响)
3. 入站必刷中的视频列表(该页面不支持使用uid、时长屏蔽，其他的不受影响)
4. 排行榜中的视频列表(大部分选项卡支持处理，除番剧、国产动漫等不需要处理)

### 用户空间主页中的动态选项卡(https://space.bilibili.com/****/dynamic)

1. 动态选项卡中的评论
2. 动态内容
3. 动态中视频标题

- 该页面中的上述2和3点仅做标题和内容的屏蔽

### 直播间(https://live.bilibili.com/****)

1. 直播间右侧弹幕评论列表

- 需要注意的是，该页面弹幕列表超过200时会删除前100条

### 直播分区页(https://live.bilibili.com/p/eden/*)

1. 直播分区页中的直播间列表
2. 该分区列表中不支持uid屏蔽

### 直播首页(https://live.bilibili.com/)

1. 该页面下的直播间列表
2. 与直播分区页不同，该页面支持uid相关屏蔽;

### 新版历史记录(https://www.bilibili.com/history)

1. 该页面下的全部、视频和专栏(直播暂不支持)
2. 不支持弹幕和播放量屏蔽，部分显示已看完的不支持时长屏蔽

### 旧版历史记录(https://www.bilibili.com/account/history)

1. 该页面下的视频列表，(不包括番剧电影等)
2. 不支持弹幕和时长以及播放量

### 首页专区(https://www.bilibili.com/v/*)

1. 该页面下的视频列表
2. 右侧的热门榜，仅支持标题和视频tag相关屏蔽

### 页面顶部搜索框的热搜

1. 该页面下的热搜项，支持模糊匹配和正则匹配

## 主面板相关功能

### 面板设置

1. 设置面板的边框颜色
2. 设置输出信息中普通消息的字体颜色
3. 设置输出信息中高亮消息的字体颜色
4. 恢复上述颜色设置
5. 隐藏主面板开关按钮，勾上则隐藏，取消勾选则显示，勾选之后刷新页面会保留状态
6. 展开或关闭主面板默认快捷键为键盘上tab键上的~键

### 规则管理

1. 添加、移除、修改相关屏蔽规则
2. 规则的导入导出、导入方式为覆盖式导入
3. 旧版本规则覆盖导入
4. 规则信息的数量查看

### 其他参数过滤

- 非规则管理中相关规则之外的参数过滤

1. 支持设置视频的时长、弹幕量、播放量、评论数等参数进行过滤
2. 支持设置评论区用户等级的过滤
3. 参数信息的数量查看

### 兼容设置

1. 首页屏蔽适配<a href="https://greasyfork.org/zh-CN/scripts/443530-bilibili-gate" target="_blank">Bilibili-Gate脚本(
   bilibili-app-recommend)</a>
2. 兼容[BewlyBewly插件](https://github.com/BewlyBewly/BewlyBewly)
3. 弃用旧版评论区处理，勾选之后，相关评论区的处理会使用新版评论区处理方式
4. 注意，除非安装了下面的脚本或插件，不要启用相关兼容，否则可能会导致脚本部分功能失效

### 与Bilibili-Gate脚本，原bilibili-app-recommend脚本兼容说明

1. 适配使用了该脚本对首页视频列表内容的展示，支持相关屏蔽操作。
2. 默认不启用，需要手动启用，如需启动，请在主面板选项卡【兼容管理】勾选"首页屏蔽适配Bilibili-Gate脚本(
   bilibili-app-recommend)“。

### BewlyBewly插件

1. 如使用了BewlyBewly插件，请在主面板选项卡【兼容管理】勾选"兼容BewlyBewly插件“。

#### bewlyBewly插件兼容说明

1. 下面选项卡中对应的视频列表

- 首页
    - 个性推荐
    - 热门视频
    - 排行(除下面之外的)
        - 番剧
        - 综艺
        - 电视剧
        - 中国动画
- 历史记录

### 缓存个tags管理

1. 每个域名下的tags数据不同
2. 缓存的目的是减少请求次数，提高脚本的运行效率和检查视频tags的速度
3. 支持导出对应域名下的tags数据
4. 支持追加导入tags数据
5. 支持清空对应域名下的tags数据

### 输出信息

1. 输出信息中显示屏蔽信息
2. 清空消息输出框

### 关于和问题反馈

1. 相关反馈地址
2. 作者其他油猴脚本地址

## 快捷屏蔽

- 大部分的视频列表，鼠标放在列表中视频标题部分时会显示屏蔽按钮，移开鼠标则隐藏
- 大部分评论列表，鼠标放在评论区评论部分时会显示屏蔽按钮，移开鼠标则隐藏
- 直播间右侧弹幕评论列表，鼠标移到评论部分时显示屏蔽按钮，反之隐藏
